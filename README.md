## Setting up the project 
# 1. Clone the repo
git clone https://github.com/SHREEESH7073/fund-pathway-app.git
cd fund-pathway-app

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev


## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

Fund Pathway — Loan Utilization Tracking System
A dual-portal web app built for Smart India Hackathon (SIH) that connects loan beneficiaries with bank employees for real-time tracking, document verification, and transparent communication.
Features
Beneficiary Portal

Loan dashboard with sanctioned/utilized/remaining amount
Upload verification documents
Verification timeline & status tracking
Map view, notifications, support chat

Bank Employee Portal

Dashboard with stats (customers, active loans, pending verifications)
Add & manage customers
Update loan progress stages
Send notifications to beneficiaries
